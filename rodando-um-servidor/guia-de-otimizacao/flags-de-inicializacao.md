# Flags de inicialização

[Minecraft Vanilla e servidores de Minecraft na versão 1.20.5+ requerem Java 21 ou maior](https://docs.papermc.io/java-install-update). A Oracle mudou seu licenciamento e não há mais um motivo convincente para obter o java deles. Fornecedores recomendados são [Adoptium](https://adoptium.net/) e [Amazon Corretto](https://aws.amazon.com/corretto/). Implementações alternativas de JVM, como OpenJ9 ou GraalVM, podem funcionar, no entanto, elas não são suportadas pelo Paper e são conhecidas por causar problemas, portanto, não são recomendadas no momento.

Seu coletor de lixo pode ser configurado para reduzir picos de lag causados ​​por grandes tarefas do coletor de lixo. Você pode encontrar flags de inicialização otimizados para servidores Minecraft [aqui](https://docs.papermc.io/paper/aikars-flags). Lembre-se de que esta recomendação não funcionará em implementações JVM alternativas. É recomendável usar o gerador de sinalizadores de inicialização [flags.sh](https://flags.sh) para obter as flags de inicialização corretas para o seu servidor
